
# Releasing ACC

- check the data
  ```shell script
  acc dump
  ```
- write the bib
  ```shell script
  acc bib
  ```
- update GBIF mappings for species
  ```shell script
  acc gbif
  ```
- compute GBIF coverage
  ```shell script
  acc coverage
  ```
- compute "distance from homo sapiens" order
  ```shell script
  acc order
  ```

Then commit and push changes.
